# Notebooks

Jupyter notebooks for SpatialAgent demos and case studies.

## Demos

```
notebooks/
└── demos/
    ├── cadinu_v6-hc.ipynb                  # Main agent demo (full pipeline)
    ├── cadinu_subagent_demo.ipynb          # Deep research & verification subagents
    ├── annotation.ipynb                    # Cell type annotation workflow
    ├── panel_design.ipynb                  # Gene panel design workflow
    └── cell_cell_communication.ipynb       # Cell-cell communication workflow
```

## Quick Start

1. **`demos/cadinu_v6-hc.ipynb`** - Main demo showing full analysis pipeline
2. **`demos/annotation.ipynb`** - Cell type annotation workflow
3. **`demos/panel_design.ipynb`** - Gene panel design workflow
4. **`demos/cell_cell_communication.ipynb`** - Cell-cell communication analysis
